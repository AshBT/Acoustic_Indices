## Installation

This library needs:

 * Python
 * Numpy
 * Scipy
 * Matlplotlib

 On Mac OS X, you can use homebrew (http://brew.sh/). To install it simply launch a terminal and enter:
 $ ruby -e "$(curl -fsSkL raw.github.com/mxcl/homebrew/go)"
Then install python
$ brew install python
and the pyhon package
$ pip install numpy scipy matplotlib
